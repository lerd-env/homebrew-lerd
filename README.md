# homebrew-lerd

Homebrew tap for [Lerd](https://lerd.sh), the Herd-like local PHP development
environment for Linux and macOS.

```bash
brew install lerd-env/lerd/lerd
lerd install
```

Recent Homebrew versions block third-party taps until trusted, so you may need
`brew trust lerd-env/lerd` first. Update later with `brew upgrade lerd`.

The `lerd.rb` formula here is generated on each release; see the
[main repo](https://github.com/lerd-env/lerd) and the [docs](https://lerd.sh)
for everything else.
